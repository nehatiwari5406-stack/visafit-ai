# VisaFit AI

VisaFit AI is a job-fit and sponsorship-risk analyzer designed for international MBA students.

The tool helps users evaluate job descriptions by analyzing:
- Resume fit
- Sponsorship risk
- Functional role alignment
- Job description keywords
- Resume gaps
- Application priority
- Outreach messaging

## Problem

International students often waste time applying to roles without knowing whether the role matches their profile, whether the employer may support work authorization, or whether the job description aligns with their career goals.

VisaFit AI helps make the job search more strategic by turning a job description into a structured application decision.

## What the App Does

A user can paste a job description and a profile summary. The app then returns:

1. Resume Fit Score
2. Sponsorship Risk Signal
3. Application Decision
4. Role Classification
5. Key JD Keywords
6. Resume Gaps
7. LinkedIn Outreach Message
8. Resume Improvement Suggestions

## Built With

- Python
- Streamlit
- Rule-based logic
- AI-assisted development using Codex

## How to Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py

## Project Goal

The goal of VisaFit AI is to help international MBA candidates make better job-search decisions by combining career strategy, recruiting logic, and AI-assisted analysis.
